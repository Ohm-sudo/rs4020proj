Measuring ChatGPT Efficiency

The purpose of this project is to evaluate the efficiency of ChatGPT in answering questions from domains such as History, Social Science, and Computer Security.

Research Approach

  • Created a GitHub repository to host web content.
  • Synced changes using GitHub Desktop.
  • Used Node.js, Express.js, Mongoose for dynamic content.
  • Utilized MongoDB Atlas for cloud-based data storage.
  • Integrated a ChatGPT API for database interaction.

Dataset Overview

  • Domains: History, Social Science, Computer Security.
  • 50 questions per domain, stored in a MongoDB database called ChatGPT_Evaluation.
  • Each question includes fields such as _id, question, options (A, B, C, D), correctAnswer, and chatGPTResponse.
